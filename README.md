## Install dev environment

### Yarn

```shell
brew update
brew install yarn
```

### Dependencies

```shell
yarn install
```

## Run

```shell
yarn run dev
```

## Lint

```shell
yarn run lint:watch
```

## Test

```shell
yarn run test
```

## Start local DynamoDB

```shell
yan run dynamo
```

> You can open shell at http://localhost:8000/shell/

## Update dependencies

```shell
yarn run npm-check
```
